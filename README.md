## SegForge

**SegForge** is a web-based experimental tool designed to assist in the identification of AI-generated imagery. Instead of relying solely on binary "AI or Human" labels, SegForge utilizes **Large Language Models (LLMs)** to provide descriptive analysis, helping users identify potential artifacts and inconsistencies common in AI-generated content.

This project is currently under active development. Some features are partially implemented or in the architectural planning phase.
